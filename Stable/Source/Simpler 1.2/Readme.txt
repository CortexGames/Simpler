Where to place a class in Simpler?

If it uses Box2d it goes in Physics
If it is interactive using the mouse / keyboard it goes in UI
If it connects to the internet it goes in Internet
If it is a collective of code, it goes in extension
If it is directly related to the game engine it goes in Game
If it is an external input device or functionality it goes in External
If it doesn't fit into these categories it goes in Display
